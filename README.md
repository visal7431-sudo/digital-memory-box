## Introduction
The Digital Memory Box is a web-based application designed to securely store, organize, and preserve personal memories such as photos, videos, and notes. It acts as a digital time capsule for long-term memory preservation.

## Problem Statement
People store memories across multiple devices, leading to poor organization, data loss, and security issues. There is a need for a centralized and secure platform for memory storage.

## Features
- Secure user authentication
- Upload photos, videos, and notes
- Organized memory management
- Cloud-based storage
- User-friendly interface

## Tools & Technologies
- Frontend: HTML, CSS, JavaScript
- Backend: Python (Flask/Django)
- Database: MySQL / MongoDB
- Version Control: GitHub

## Future Scope
- AI-based photo tagging
- Voice memory support
- Mobile application
